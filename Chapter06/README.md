# Source Code

This repository contains the core Flutter source code for the app.

To keep the project lightweight and focused, only the following are included:

- `lib/`
- `pubspec.yaml`

The remaining Flutter project files and folders are intentionally excluded.

---

## What is included

- `lib/` → the full Dart source code of the app
- `pubspec.yaml` → project dependencies and Flutter configuration

## What is not included

This repository does **not** include generated or platform-specific folders such as:

- `android/`
- `ios/`
- `web/`
- `windows/`
- `linux/`
- `macos/`
- `.dart_tool/`
- `build/`
- `.idea/`
- `.vscode/`

These can be regenerated easily using Flutter tools.
